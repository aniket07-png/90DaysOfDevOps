What is the difference between git add and git commit?
git add is use to push the file at staged and 
git commit is use to push the file at tracked stage.

What does the staging area do? Why doesn't Git just commit directly?
Staging area is use to review changes before committing and choose which changes to commit.
we may change multiple files but only want to commit some of them.

What information does git log show you?
It is shows the git commit history, author,date of commit,etc.

What is the .git/ folder and what happens if you delete it?
.git/ folder is the hidden folder which contains files about hooks,config,etc
if you delete this folder your directory will no longer git repository.

What is the difference between a working directory, staging area, and repository?
working directory is where you can make and edit files
staging area is where you can choose which file or changes you want to commit
repository is where you already committed the codes or changes to your git repository.
