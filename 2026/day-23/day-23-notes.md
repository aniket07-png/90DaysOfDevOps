What is a branch in Git?
Branch gives you a separate part where you can deploy changes and other things without disturbing the stability of main branch.

Why do we use branches instead of committing everything to main?
We use branches instead of committing everything to main because we need work tree clear as well as commit history if we commmit everything in main the working tree and commit history become messy.

What is HEAD in Git
Head point the current working branch.
