What is a branch in Git?
Branch gives you a separate part where you can deploy changes and other things without disturbing the stability of main branch.

Why do we use branches instead of committing everything to main?
We use branches instead of committing everything to main because we need work tree clear as well as commit history if we commmit everything in main the working tree and commit history become messy.

What is HEAD in Git
Head point the current working branch.

What is the difference between clone and fork?
We use clone to make copy of repository from remote to local
and fork is use to copy repository from other remote repository to your remote repository 

When would you clone vs fork?
If i want repository in my local i will use clone but if i want other repository to my remote i use fork.

After forking, how do you keep your fork in sync with the original repo?
We use sync fork to keep our fork in sync with the original repository.
